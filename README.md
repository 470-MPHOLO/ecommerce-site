# ShopEasy E-commerce Website

A fully functional e-commerce website similar to Takealot, with admin panel, product management, and cart functionality.

## Features

### Customer-Facing Features
- Product browsing by category
- Product search functionality
- Shopping cart with quantity management
- Contact options (Email, WhatsApp, Phone)
- Responsive design for mobile and desktop

### Admin Features
- Hidden admin access via settings gear icon
- Product management (add, edit, delete)
- Secure password-protected admin panel
- No login system required (single password)

### Security Features
- Admin panel password protection
- Local storage for data persistence
- Input validation and sanitization
- Secure session management

## Setup Instructions

1. **Clone the repository** or download all files
2. **Open `index.html`** in a web browser
3. **Default admin password**: `admin123` (change in code for production)
